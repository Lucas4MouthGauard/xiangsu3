# PumpAlien: 真相揭露计划 🛸👽

> **当Crypto成为外星人的试验场，真相即将揭露**

## 🌟 项目概述

PumpAlien是一个基于React + TypeScript + Vite构建的交互式真相揭露平台，讲述了一个关于外星人PumpAlien如何利用Crypto交易收集能量，最终实现其终极计划的惊悚故事。

### 🎭 核心故事线

**"人类以为Alon是PumpFun的创始人，但真正的Alon在很久以前就被外星人绑架。取而代之的是一个伪装者，一个来自未知星系的存在，名字叫PumpAlien..."**

- **第1章**: 消失的Alon - 人类以为Alon是PumpFun的创始人
- **第2章**: 宇宙绑架案 - 真正的Alon被外星人绑架
- **第3章**: 伪装者降临 - PumpAlien穿上了人类的外壳
- **第4章**: Crypto试验场 - 每一笔交易都为PumpAlien提供力量
- **第5章**: 能量觉醒 - 当能量充满，地球将成为他的舞台

## ✨ 主要特性

### 🎮 交互式故事体验
- **章节式故事揭示**: 逐步解锁真相，体验沉浸式叙事
- **动态内容加载**: 根据用户进度动态显示内容
- **打字机效果**: 模拟真实的信息传输过程

- **动态价格波动**: 模拟真实市场的价格变化

### 📅 时间线追踪
- **事件时间线**: 从2023年到2025年的关键事件
- **影响评估**: 每个事件对人类文明的影响等级
- **详细分析**: 深入解析每个事件的背景和后果

### 🚨 真相揭示系统
- **渐进式揭露**: 随着能量收集逐步揭示真相
- **最终警告**: 当能量达到100%时触发终极真相
- **行动选择**: 提供多种应对策略选择

## 🚀 快速开始

### 环境要求
- Node.js 18.0.0 或更高版本
- npm 9.0.0 或更高版本

### 安装和运行

#### 方法1: 使用启动脚本（推荐）

**macOS/Linux:**
```bash
chmod +x start.sh
./start.sh
```

**Windows:**
```cmd
start.bat
```

#### 方法2: 手动安装

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

```typescript
export const pumpAlienStory = {
  title: "自定义标题",
  chapters: [
    // 自定义章节
  ]
}
```

### 样式定制
在 `src/styles/index.css` 中修改CSS变量：
```css
:root {
  --primary-color: #your-color;
  --secondary-color: #your-color;
}
```
