# 🌌 费米悖论：PumpAlien的宇宙探索

## 项目简介

这是一个以**费米悖论**为主题的交互式网页项目，结合了真实的科学数据、好莱坞级别的剧情设计和现代化的用户体验。用户将扮演一个服用了神秘药物💊Pump的外星人，探索宇宙中最大的谜题之一。

## 🚀 项目特色

### 科学准确性
- **德雷克方程**：基于真实的天文学数据
- **SETI项目数据**：包含真实的信号搜索频率
- **系外行星统计**：最新的天文发现数据
- **费米悖论假说**：基于科学研究的理论

### 好莱坞级剧情
- **PumpAlien主角**：独特的外星人角色设定
- **多线剧情**：科学探索、哲学思考、宇宙冒险
- **沉浸式体验**：星空背景、动画效果、交互元素

### 现代化设计
- **响应式布局**：支持各种设备
- **CSS动画**：流畅的过渡效果
- **交互式元素**：点击、悬停、触摸手势
- **视觉特效**：星空背景、发光效果、渐变色彩

## 📱 使用方法

### 基本操作
1. 点击"开始探索"按钮
2. 阅读费米悖论的科学解释
3. 选择探索路径（科学/哲学/冒险）
4. 体验不同的交互内容
5. 揭示宇宙真相

- **点击卡片**：查看假说详情

### 冒险模式
- **扫描星系**：发现文明遗迹

- **星际旅行**：探索新区域
- **能量管理**：策略性决策

## 🔬 科学数据来源

### 德雷克方程参数
- R* = 7 (银河系恒星形成率/年)
- fp = 0.5 (有行星的恒星比例)
- ne = 2 (宜居行星数量)
- fl = 0.1 (生命出现概率)
- fi = 0.01 (智能生命概率)
- fc = 0.1 (文明通信概率)
- L = 1000 (文明寿命/年)

### SETI搜索频率
- 1420 MHz (氢线)
- 1665 MHz (羟基线)
- 22 GHz (水线)

### 系外行星统计
- 已确认：5000+
- 宜居带：约50个
- 类地行星：约12个
